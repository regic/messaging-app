# README

Requirements:

* Ruby version - 2.3.1

* Rails version - 5.1.1

* Database - PostgreSQL 9.x

* To run the specs (you may need to install `bundler`) type in terminal:

  * `bundle install`

  * `rails db:migrate`

  * `rspec`
